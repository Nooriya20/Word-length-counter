# Word Length Finder 📝

A web app that maps words to their lengths using dictionary logic.

## How It Works
- Type any words separated by spaces
- Click Find Lengths
- See each word mapped to its length
- Shows longest and shortest word

## Features
- Clean dark interface
- Animated results
- Shows word count summary
- Originally built in Python, converted to a web app

## Technologies Used
- Python (original logic)
- HTML
- CSS
- JavaScript
